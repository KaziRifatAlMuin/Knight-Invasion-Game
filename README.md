# Knight-Invasion-Game
2 AI Player Game
- Minimax (Kazi Rifat Al Muin)
- Fuzzy (Dipta Chowdhury)

## AI Agents

- **agent_red** (Red Knight) — Minimax AI implemented by Kazi Rifat Al Muin
- **agent_blue** (Blue Knight) — Fuzzy AI implemented by Dipta Chowdhury

Players use color-themed agents: Red (optimal Minimax) vs Blue (fuzzy heuristics).
